# Yatube API

REST API для социальной сети Yatube. Позволяет работать с постами, комментариями, группами и подписками.

## Основные возможности

- Создание, редактирование и удаление постов
- Комментирование постов
- Работа с сообществами (группами)
- Система подписок на авторов
- JWT-аутентификация
- Пагинация и поиск

## Технологии

- Python 3.7+
- Django 3.2+
- Django REST Framework
- Simple JWT
- SQLite
